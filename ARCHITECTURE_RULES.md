# GIOS Architecture Rules

The GIOS system architecture is permanently fixed.

AI agents are not allowed to modify the system architecture.

Allowed directories:

/core
/engines
/intelligence
/interface
/infrastructure

AI agents are NOT allowed to:

- create new architecture folders
- rename core directories
- move system modules outside these directories
- duplicate engines

Architecture changes require human authorization.

System Architect: Human Governance
